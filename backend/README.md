# Backend

This backend is responsible for interacting with the Pokémon Trading Card Game API to provide necessary data and functionality for the frontend. It handles various operations, including fetching card information, managing user data, and providing endpoints for different functionalities.

## Setup Instructions
1. Clone the repo: `git clone https://github.com/<owner>/pokemon-tcg.git`
2. Navigate to the backend directory: `cd pokemon-tcg/backend`
3. Install the necessary dependencies: `npm install`
4. Start the server: `npm start`

## Environment Variables
Make sure to set the following environment variables:
- `API_KEY`: Your API key for accessing the Pokémon TCG API.
- `DB_URL`: The URL for your database.
